# 结束

这篇笔记和示例到这里就结束了。

当然，你的学习还没有结束，毕竟这里没有提及、深入很多东西，比如部署应用包时的基本 URL，CSS Modules 支持，MPA 的实践，SSR 支持，`webpack` 配置封装，使用 `webpack` 构建 `vue` 应用，更复杂的 TypeScript 实践等等。

真正开发的时候，建议直接使用 cli。

另外，也推荐你去学习 `rollup` 和 `vite`。
